## Learning Path

Week 3 has 6 topics + 1 assignment.

Work through the topics in order — each one narrow step that sets up the next:

```
Backend Foundations (How the Web Really Works + What is a Backend)
    |
    |  You need to understand requests, responses, and HTTPS before any framework makes sense
    v
Django Basics (MTV, Project Setup, Models, Admin)
    |
    |  Learn the Django framework and why it is used
    v
PostgreSQL + the ORM (Real Database, Python <-> SQL bridge)
    |
    |  A real database server other than the Django default SQLite
    v
Django REST Framework (Turning Django into a JSON API)
    |
    |  React can't read Django templates. It needs JSON. DRF is how Django speaks JSON
    v
Auth: JWT & OAuth (Securing the API, Login with Google/GitHub)
    |
    |  An open API is a liability. This is how you control who can do what
    v
Connecting React to Django (CORS, Axios, the full-stack handshake)
    |
    |  How to connect your frontend React app with Django backend
```
